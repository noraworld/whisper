# Whisper
The customized theme for [Ghost](http://github.com/tryghost/ghost/).
This theme is based on the default Ghost theme, [Casper](https://github.com/TryGhost/Casper).

## Dependencies
* `/assets/js/sidebar.js` includes author name.
